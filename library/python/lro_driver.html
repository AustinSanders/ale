class ale.drivers.lro_driver.LrocPds3Driver(file)[source]¶

Bases: ale.drivers.base.PDS3, ale.drivers.lro_driver.LrocSpice

Driver for reading Lroc labels. Requires a Spice mixin to acquire addtional ephemeris and instrument data located exclusively in spice kernels.

instrument_id¶

Returns an instrument id for uniquely identifying the instrument, but often also used to be piped into Spice Kernels to acquire IKIDs. Therefore they the same ID the Spice expects in bods2c calls.

Ignores Wide Angle for now
