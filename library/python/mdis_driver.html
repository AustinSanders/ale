class ale.drivers.mdis_driver.MdisSpice(file)[source]¶

Bases: ale.drivers.base.Spice, ale.drivers.base.Framer

MDIS mixin class for defining snowflake Spice calls. Since MDIS has unique Spice keys, those are defined here as an intermediate mixin for MDIS drivers that rely on Spice kernels.

focal_length¶

Computes Focal Length from Kernels

MDIS has tempature dependant focal lengh and coefficients need to be acquired from IK Spice kernels (coeff describe focal length as a function of tempature). Focal plane temps are acquired from a PDS3 label.

Returns:
: double

focal length in meters
