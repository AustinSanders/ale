instrument_id¶

Returns an instrument id for unquely identifying the instrument, but often also used to be piped into Spice Kernels to acquire IKIDs. Therefore they the same ID the Spice expects in bods2c calls.

: str

instrument id

: string

Path to latest metakernel file
odtk¶

The radial distortion coeffs are not defined in the ik kernels, instead they are defined in the ISS Data User Guide (Knowles). Therefore, we manually specify the codes here.

spacecraft_name¶

Spacecraft name used in various Spice calls to acquire ephemeris data.
